## windows 系统下任务栏智能隐藏工具

- 功能
  * 根据当前活动的窗口最大化隐藏任务栏
  * 为了不影响系统流畅，窗口切换后如果用户正在操作键盘或鼠标，则任务栏不会切换状态，待停止一定时间后再进行切换。
